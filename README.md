# WebEss — Simple Static Pages

A tiny static website demonstrating basic HTML and CSS structure for practice.

Files
- `index.html` — Main landing page
- `page1.html`, `page2.html`, `page3.html` — Additional pages
- `style/styles.css` — Site styles

Usage
- Open `index.html` directly in your browser.
- Or serve locally for correct relative paths:


Notes
- No build tools required. Edit the HTML/CSS files and refresh the browser.

Created for a small web essentials exercise.
